## A. GM 单 Combo 视觉预览（脱离实战，无需敌人/无需 playing 态）
- [ ] 点「单 Combo 视觉预览 → steamExplosion」：蛇头处见 **白色蒸汽云 + 浅蓝冰晶碎屑 + 实心白闪核**。
- [ ] 点「electroTurret」：见 **紫电链（蛇头→3 假人）+ 节点放射紫电芒 + 蛇头紫辉光**（假人自动生成）。
- [ ] 点「burningBarrage」：见 **橙色燃烧镖飞向假人 + 命中橙焰爆点 + 假人身红橙燃烧环**。

## B. 实战三 combo 触发一眼可区分（飘字 + 伤害数字 + 演出）
- [ ] 解锁 fire+ice → 周期见蒸汽白爆（白闪核+冰晶），伤害飘字 **「💥蒸汽 -N」暖橙**。
- [ ] 解锁 bolt+lightning（**无 fire**）→ 弹射子弹命中触发 **紫电链 + 节点电芒**；**该下 bolt 命中飘「飞镖 -N」青**（飞镖归飞镖，P1 修复：不再误贴「电磁」），**连锁命中飘「⚡电磁 -N」紫**（独立来源 `src='electro'`，不写死「闪电」）。飞镖与电磁**两笔数字同时可见、不混**。
- [ ] 仅 lightning（无 bolt）→ **蓝白电链**，飘字 **「闪电 -N」蓝白**（与电磁紫区分）。
- [ ] **⚡电磁 vs 基础闪电「一眼区分」增强（B-4 ①·纯表现·并入 B-4）**：
  - 电磁（fx:electroarc）：电链**明显更粗**（~5px vs 基础 2px）+ 每个节点**多分叉放射紫电芒** + 命中点**紫色残留辉光(~0.2s afterglow)** + 存活更久（~0.34s）。
  - 基础闪电（fx:lightning）：保持**细 / 快 / 蓝白 / 单链 / 无残留辉光**，靠简洁与电磁形成对比。
  - 验收：实战同屏出现两者时，**不靠颜色也能一眼认出哪个是电磁**（粗+分叉+残留 vs 细快无残留）。
- [ ] **P1/P2 修复验收（并入 B-4）**：
  - **飞镖归位**：bolt+lightning combo 下，bolt 命中飘 **「飞镖 -N」青**、电磁连锁飘 **「⚡电磁 -N」紫**，**两笔同时可见、不混**（P1：曾误把每次飞镖伤害全贴「电磁」掩盖真连锁，已修）。
  - **电磁稳定触发**：电磁 combo 按 **~0.5s** 节奏稳定触发连锁（不再十几秒才一次；P2：`timer.electro -= dt` 移出 `return` 后、每帧推进，`cooldownSec=0.5s` 真生效；bug 修复非数值改动，不写真源）。
  - **粗弧看得清**：因触发正常，粗弧+节点分叉+命中残留辉光现在**看得清、不再一闪而过**。
  - **无 lightning 时纯飞镖正常**：只升 bolt（无 lightning）→ 飞镖照常飘「飞镖」、无电磁连锁（`foundCombo.electroTurret` 不激活）。
  - **灼烧仍飘「🔥灼烧」**：burningBarrage combo 下 bolt 命中仍带 `burning` 橙标，未受 P1 影响。
  - _电磁 DPS 是否过强本轮只记手感、不调数值（留 ③）；若实测偏强，下一轮按 §七 回写真源 `cooldownSec`/`damageMul`_。
- [ ] fire+bolt 灼烧弹幕（**无 lightning**）→ 橙镖 + 敌身火环；**该下 bolt 命中即飘「🔥灼烧 -N」橙色**（不再飘「飞镖」），后续引燃跳 **「🔥灼烧 -N」红橙飘字**（与火墙区分）。
- [ ] **DOT 分源分离（B-4 ① 同范畴·纯表现，已并入 B-4 提交别单开）**：
  - ① 火墙与灼烧引燃 **各自独立数字 + 独立标签**（`🔥火墙 ` 橙 #ff9a3c / `🔥灼烧 ` 红橙 #ff5a2c）。
  - ② 同一敌人**同时中火墙+引燃**→ **两个独立数字分别飘、互不相加、标签不串**（橙 / 红橙）。
  - ③ 敌人**死亡时各来源残留 DOT 分别 flush**（各源独立带标签出字，不丢不混、无「0」刷屏）。
  - ④ **对象池复用 `dotMap={}` 复位、无残留串味**（连续击杀多敌不串源）。
  - _节奏说明：分源后各源需**各自 ≥ `DOT_TEXT_MIN`** 才飘字，单源闪字节奏略变属显示层无妨。_
- [ ] fire+bolt+lightning 三者全开 → bolt 命中飘「🔥灼烧」(burning 优先)、连锁飘「⚡电磁」，两 combo 名同时可见且各异。
- [ ] 三 combo 颜色 / 图标 / 飘字各异，肉眼可辨。
